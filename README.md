# SaarFFmpeg

.Net Core的FFmpeg库。

功能
------
* 音视频解码(audio&video decoder)
* 音视频编码(audio&video encoder)
* 音视频重采样(audio&video resampler)
* 音视频封装(audio&video remuxer)
* DSP

需要DLL
------
* avutil-56.dll
* avcodec-58.dll
* avformat-58.dll
* swscale-5.dll
* swresample-3.dll
* avdevice-58.dll
* avfilter-7.dll
* postproc-55.dll

[下载DLL](http://ffmpeg.zeranoe.com/builds/)，下载页面里选择Shared。
